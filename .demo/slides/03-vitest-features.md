## Vitest Features
- It's integrated with Vite. It's very easy to get started by installing and running Vitest.
- It has a fast watch mode.
- Out of the box, it supports testing JSX, TypeScript.
- Vitest offers excellent support for snapshot testing.
- Finally, Vitest offers concurrency, which means that it's able to execute multiple tests in parallel at the same time.
