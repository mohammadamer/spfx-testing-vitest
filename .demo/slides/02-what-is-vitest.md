# What is Vitest?
- Vitest is a testing library which is inspired by Jest and closely integrated with the Vite ecosystem.

# Vite or Vit?
- Vite, or Vit, is a scaffolding tool, which emphasizes very fast reloads. Vitest is to Vite as Jest is to create‑react‑app.
