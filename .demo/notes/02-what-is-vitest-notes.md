---

1. What is Vitest?
Vitest is fundamentally a testing library.
It is heavily inspired by Jest, meaning the syntax and structure will feel very familiar to many of you.
Crucially, it is closely integrated with the Vite ecosystem. This integration is its core differentiator.

2. The Ecosystem: Vite's Role
What is Vite? It's a scaffolding tool and a modern front-end build tool.
Vite emphasizes very fast reloads—this is its superpower.
Simple Analogy: Think of Vite as the modern equivalent of create-react-app.
The Relationship: Vitest is to Vite as Jest is to create-react-app. This clearly defines its role in the ecosystem.

3. Key Features and Advantages
Speed is King: Just like Vite, Vitest focuses on quick test execution.
It's designed with numerous built-in features to assist in fast test runs and improve developer workflow.
Full-Featured: This is not a lightweight tool. Vitest is a full-featured testing library.
It comes with many essential tools built-in, including:
Snapshots
Coverage
User Interface (UI) for debugging
And much more!
